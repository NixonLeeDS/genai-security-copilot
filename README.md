# GenAI Security Copilot

A basic local GenAI project that analyzes cloud security related inputs and returns
human readable risk explanations and remediation guidance.

## Current Scope
- Local only development
- Mock LLM responses
- Advisory analysis only
- No automated actions

## Tech
- Python
- FastAPI (local)
- Prompt based GenAI logic

This project is built to demonstrate practical application of GenAI concepts
and will be extended to cloud services later.
